# 项目简介

本项目是基于多窗联合谱分析与动态自适应深度学习的旋转机械故障智能诊断系统。其主要用途是对旋转机械设备的故障进行高效、准确的识别和诊断。系统功能模块包括数据预处理、特征提取、故障识别及结果展示。技术特点在于采用多窗联合谱分析技术以增强信号解析能力，结合动态自适应深度学习算法，实现复杂工况下的故障诊断。此技术具有较强抗干扰性和自适应能力，适用于多种旋转机械设备的故障诊断应用。


## 核心功能
- 数据预处理：对旋转机械设备信号进行清洗和标准化处理
- 特征提取：利用多窗联合谱分析技术提取设备故障特征
- 故障识别：采用动态自适应深度学习算法进行智能故障诊断
- 结果展示：直观呈现故障诊断结果和分析报告

## 适用场景
适用于各类旋转机械设备（如电机、轴承、齿轮箱等）的故障诊断，能够在复杂工况环境下实现高效、准确的故障识别，帮助用户及时了解设备运行状态，预防潜在故障。

## 技术特点
- 多窗联合谱分析技术：增强信号解析能力，提高特征提取准确性
- 动态自适应深度学习算法：根据工况变化自动调整诊断模型
- 强抗干扰性：在噪声环境下保持稳定诊断性能
- 广泛适用性：支持多种类型旋转机械设备的故障诊断
